Who Am I

Hi, I’m Ollie. I run Obford - a solo consultancy focused on making complex IT work seamlessly for teams. I’ve spent over 5 years solving tough problems across hardware, software, networking, and data - for industries where downtime simply isn’t an option.

My background spans both large studios like Playground Games, as well as smaller, more bespoke tech environments. That means I can bring enterprise-level solutions to teams of any size, delivering results that stick.

Some of my work includes:

  • Playground Games - Running studio IT and creating custom tools to streamline workflows.
  • Baker Hughes - Designing industrial-grade, failure-resistant server solutions for remote operations.
  • Beyond - Using data analysis to guide decisions on new store locations.
  • Terra / Wortmann - Supporting hardware systems and training teams on new server and cloud platforms.

I’m all about providing clear advice, building systems that are easy to maintain, and making technology work reliably for the people who use it.
